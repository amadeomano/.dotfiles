# Intro
This package provides a thin layer of abstraction over web API URL functions to ensure they are type-safe.

# Installation
Add the following to your `package.json`:
```json
"@personio-web/payroll-util-navigation": "workspace:*"
```

# Usage
To make the web API's search parameters type-safe, use the `createParamHandlers` function and pass a list of parameters you want to manage:

```ts
// 1. Import the package
import { createParamHandlers } from '@personio-web/payroll-util-navigation';

// 2. Create a constant array containing the search parameters you want to manage
const managedParams = [
  'legalEntityId',
  'payGroup',
] as const;

// 3. Call `createParamHandlers` with your params and export its type-safe CRUD functions
export const { getParams, setParam, deleteParam } = createParamHandlers(managedParams);

// 4. Optionally, export `createUrl` and `commitNavigation` as well
export {
  createUrl,
  commitNavigation,
} from '@personio-web/payroll-util-navigation';
```

# Usage of the Type-Safe Methods
With the methods you exported above, you can perform the following actions:

```ts
import { getParams, setParam, deleteParam, createUrl, commitNavigation } from './aboveFile.ts';

// 1. `getParams` returns an object with your params as keys
const { legalEntityId, payGroup } = getParams(nextRouter.query);

// 2. `createUrl` builds a new URL object either from scratch or while keeping the current route's search params
const newUrl = createUrl(nextRouter, true); // Clears all the search params
const newUrl = createUrl(nextRouter, false); // Keeps the search params

// 3. Set or delete parameters from a URL object
setParam(newUrl, 'legalEntityId', 123);
deleteParam(newUrl, 'legalEntityId');
```
